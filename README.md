# Full Stack Open

Repositório com os exercícios do curso [Full Stack Open](https://fullstackopen.com/ptbr/) da Universidade de Helsinki.

## Aplicações em produção

- **Phonebook (Parte 3 em diante):** https://phonebook-backend-5i2j.onrender.com

> O serviço fica em sleep após 15 minutos de inatividade (free tier do Render). A primeira request após esse período pode levar até 1 minuto para acordar.

## Estrutura

```
part0/                       diagramas Mermaid
part1/                       React fundamentos (courseinfo, unicafe, anecdotes)
part2/                       Comunicação com servidor (courseinfo, phonebook, countries)
part3/phonebook-backend/     Backend Node.js + Express servindo o frontend do phonebook
```
